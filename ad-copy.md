# CCFM 광고 카피 생성기

사용자가 $ARGUMENTS 로 원본 카피 또는 제품/캠페인 정보를 전달합니다.
예시: `/ad-copy 봄맞이 신상 20% 할인, 지금 바로 만나보세요`

아래 파이프라인을 **반드시 순서대로** 실행하세요.

---

## Step 1: 입력 분석

사용자 입력에서 아래를 추출합니다:

| 항목 | 추출 |
|------|------|
| 원본 카피 | 변형할 기존 카피 (있으면) |
| 제품/서비스 | 무엇을 광고하는지 |
| 타겟 | 누구를 대상으로 |
| USP | 핵심 차별점/혜택 |
| 채널 | 메타/네이버/카카오/구글/인스타 등 |
| 톤 | 감성/직설/질문/긴급/유머 등 |
| 글자 수 | 채널별 제한 (아래 참조) |
| A/B 테스트 | 필요 여부 |

### 채널별 글자 수 가이드
| 채널 | 헤드라인 | 본문 | 비고 |
|------|---------|------|------|
| 메타 광고 | 25자 | 90자 | Primary text + Headline |
| 구글 검색광고 | 30자 | 90자 (설명 2줄) | 반응형 검색광고 기준 |
| 네이버 SA | 15자 | 45자 | 파워링크 기준 |
| 카카오 비즈보드 | 15자 | 45자 | 모바일 기준 |
| 인스타 피드 | - | 200자 내외 | 해시태그 포함 |
| 인스타 릴스 | - | 100자 | 짧을수록 좋음 |
| 카카오 알림톡 | - | 1000자 | 긴 본문 가능 |
| 이메일 | 40자 | 제한 없음 | 제목이 핵심 |

## Step 2: 카피 생성

### 필수 출력 구조

각 채널 × 각 톤에 대해 아래 형식으로 생성합니다:

```
## {채널명}

### A안 ({톤})
- 헤드라인: {글자수 표시}
- 본문: {글자수 표시}
- CTA: {버튼/링크 텍스트}
- 추천 비주얼: {이미지 방향 1줄}

### B안 ({톤})
- 헤드라인: {글자수 표시}
- 본문: {글자수 표시}
- CTA: {버튼/링크 텍스트}
- 추천 비주얼: {이미지 방향 1줄}
```

### 톤 변형 규칙

| 톤 | 특징 | 예시 |
|----|------|------|
| **감성형** | 공감, 스토리, 감정 | "당신의 하루에 작은 선물" |
| **직설형** | 숫자, 혜택, 직접적 | "20% 할인, 오늘까지" |
| **질문형** | 호기심, 공감 질문 | "올 봄, 뭐 입지?" |
| **긴급형** | 시간 제한, FOMO | "남은 수량 23개" |
| **유머형** | 위트, 밈, 친근 | "지갑이 열리는 소리가 들려" |
| **증거형** | 사례, 리뷰, 데이터 | "2만명이 선택한 이유" |

### 카피 품질 규칙
- 글자 수 엄격 준수 (각 카피 옆에 글자 수 표기)
- 주어 생략 선호 (한국어 광고 특성)
- 동사로 끝나기 (행동 유도)
- 구체적 숫자 포함 (추상적 표현 금지)
- 검증된 카피 프레임워크 활용:
  - AIDA (Attention-Interest-Desire-Action)
  - PAS (Problem-Agitate-Solve)
  - BAB (Before-After-Bridge)
  - 4U (Urgent-Unique-Ultra-specific-Useful)

### A/B 테스트 안내
각 채널별 A안/B안을 제공할 때:
- A안과 B안은 **한 가지 변수만** 다르게 (톤 OR 메시지 OR CTA)
- 어떤 변수를 테스트하는지 명시
- "A안 = 혜택 중심, B안 = 감성 중심 → 어느 쪽이 CTR 높은지 테스트"

---

## Step 3: 해시태그 (인스타/SNS 채널)

SNS 채널 카피에는 해시태그 세트를 포함합니다:
- 브랜드 해시태그 2~3개
- 카테고리 해시태그 3~5개
- 트렌드 해시태그 2~3개
- 총 10개 이내

---

## Step 4: 시각화 리포트 (선택)

사용자가 요청하면 /viz 스타일로 카피 비교 시각화를 생성합니다:
- 채널별 카피 한눈에 비교
- A/B안 나란히 비교
- 글자 수 게이지

---

## Step 5: 출력 형식

### 기본 출력 (텍스트)
```
# 광고 카피 생성 결과

## 입력 정보
- 원본/제품: {내용}
- 타겟: {내용}
- 채널: {채널 목록}

## 메타 광고
### A안 (직설형) — 테스트 변수: 혜택 강조
...

## 인스타그램
### A안 (감성형)
...

## 총 생성: {N}개 채널 × {M}개 안 = {총}개 카피
```

### 시각화 출력 (요청 시)
- ~/Downloads/에 HTML 저장
- Vercel 배포
- 프로젝트명: `ccfm-adcopy-{슬러그}`

---

## 콘텐츠 규칙
- 도메인: 퍼포먼스 마케팅, 이커머스, 브랜드 마케팅
- 톤: 실무자가 바로 복사해서 쓸 수 있는 수준
- 금지: 과장 광고, 법적 문제 소지 표현, 비교 광고 시 명예훼손
- 글자 수 엄수: 각 카피에 (N자) 표기

## 응답 형식
생성 완료 시: `✅ {N}개 카피 생성 완료. 채널: {채널목록}`
